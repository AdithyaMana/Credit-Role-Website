## Overview

The Credit Roles Taxonomy Visualizer is a static frontend application built to visually showcase and provide details on the CRediT (Contributor Roles Taxonomy). It uses an interactive hexagonal grid interface to display 14 distinct roles, allowing users to select roles, view detailed descriptions, icon sets, and practical examples of their usage in academic publications.
